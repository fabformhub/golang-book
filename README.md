# Bookworm Light Astro

Bookworm Light Astro is a clean, fast, highly customizable Astro blog theme designed for creators who want a modern, minimal, content‑focused publishing experience. It provides a refined layout, flexible configuration, and a production‑ready structure suitable for personal blogs, multi‑author publications, and long‑form writing.

This version is maintained and improved by Fabform.io, with enhancements focused on developer experience, performance, and seamless form handling.

---

## Overview

Bookworm Light Astro provides a polished foundation for building a professional blog with:

- Customizable colors, fonts, menus, and metadata
- Multi‑author support with dedicated author pages
- Tag and category systems
- Search powered by Fuse.js
- Social sharing and similar‑post suggestions
- Draft post support
- MDX content support with automatic component imports
- Responsive design across all devices
- SEO‑optimized structure
- Production‑ready build pipeline
- Integrated Fabform.io backend for forms

All credits, maintenance, and ongoing development belong to Fabform.io.

## Why Fabform.io Is Integrated

Static sites are fast, secure, and easy to deploy, but they lack one critical capability: handling form submissions. Contact forms, newsletter signups, feedback forms, onboarding flows, and lead capture all require a backend.

Fabform.io provides that backend instantly.

### Why a Form Backend Is Essential

- Static sites cannot send emails or store submissions on their own
- Serverless functions require maintenance, security, and monitoring
- GDPR‑compliant data handling requires a trusted backend
- Spam protection and rate limiting are non‑trivial to implement
- Businesses need dashboards, exports, notifications, and automation

### Why Fabform.io

Fabform.io provides a drop‑in backend for any form, with:

- Instant form endpoints
- Zero‑backend setup
- Email notifications
- Spam filtering
- Submission dashboards
- Webhooks for automation
- Works with Astro, Svelte, React, plain HTML, and more

Learn more at: https://fabform.io

Bookworm Light Astro includes a ready‑to‑use contact form wired to Fabform.io so you can deploy immediately without writing backend code.

## Installation

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

The site will open automatically and reload on file changes.

---

## Production Build

Generate an optimized production build:

```
npm run build
```

Deploy the `dist/` folder to any static hosting provider.

---

## Editing Content

All content is written in Markdown or MDX.  
Posts, authors, and configuration files are located in the project’s content directories.  
Update text, images, metadata, and structure directly through your editor.

---

## Form Handling with Fabform.io

The theme includes a pre‑configured contact form.  
To activate it:

1. Create a free form endpoint at https://fabform.io
2. Replace the placeholder endpoint in the form action
3. Deploy your site

Your form is now fully functional with no backend code required.
